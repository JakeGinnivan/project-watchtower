# Project Watchtower

A unified basis for web applications.

## Features

* Webpack build
* Testing
* Cleaning
* Linting
* Build metrics
* Server middlewares and helpers
* Standalone server for client-only applications

Detailed documentation available in the [docs](./docs) folder:

* [CLI Commands](./docs/cli.md)
